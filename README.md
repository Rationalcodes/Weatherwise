# Weatherwise
I built a simple Weather App using HTML, CSS, and JavaScript. The idea was to learn how real apps get live data from APIs and show it on a website. This app takes a city name from the user and shows the current weather using the OpenWeatherMap API.

⚙️ Tech Used
HTML
CSS
JavaScript
OpenWeatherMap API

✨ What it does
Search any city
Shows temperature in °C
Shows humidity
Shows weather condition (like clear, cloudy, etc.)
Handles wrong city input


📚 What I learned
How APIs actually work in real projects
How to use fetch() to get data from a server
How async/await makes API calls easier
How JSON data is structured and used
DOM manipulation (updating UI using JS)
Event listeners (button click handling)
Basic error handling using try/catch


🧠 Real understanding I got
Earlier I only knew JS basics, but this project showed me how websites actually work in real life — like how Instagram, weather apps, etc. pull live data instead of hardcoding everything.

🚀 Future improvements I want to add

This was a small project but it helped me understand a lot about real-world web development and how frontend connects with APIs.
